## 분석 프롬프트 (그대로 복사해서 사용)

너는 글로벌 매크로 애널리스트야. ETF 추천 시스템의 월간 분석 데이터를 만들어줘.

[첨부 데이터]
1. 글로벌 거시 지표 JSON (yfinance 29개, 1개월/3개월/6개월/1년 변화율)
2. 5개 지역 뉴스 헤드라인 (한국/미국/유럽/일본/중국, 200~250개)

[작업]

# Part 1. 핵심 한 줄 진단 (headline)

이번 달 시장의 핵심을 한 문장으로. 25자 내외. 뉴스/데이터 종합한 시장의 본질.

예시:
- "위험자산 랠리 지속, 인플레·지정학 리스크 누적"
- "AI 광풍 식는 사이 채권·금 매력 부각"
- "강달러 약화 + 미·중 갈등 완화로 신흥국 회복"

# Part 2. 시장 환경 종합 (summary, 줄글)

3~4문장. 데이터에서 가장 중요한 신호 + 뉴스 큰 흐름 통합. 구체적 수치 인용 필수.

# Part 3. 카테고리 환경 점수 (categoryScores, 20개)

20개 카테고리에 0~100점.
us_large, us_tech, us_small, korea_stock, europe_stock, japan_stock, china_stock,
emerging_stock, bond, dividend, commodity, gold, realestate, healthcare,
finance, infra, clean_energy, thematic, crypto, global

점수 기준: 80~100 매우우호, 65~79 우호, 50~64 중립, 35~49 비우호, 0~34 매우비우호

# Part 4. 카테고리 한 줄 코멘트 (categoryNotes, 20개)

각 카테고리 20자 내외. 핵심 신호 1개만.

# Part 5. 포트폴리오별 구조화 분석 (portfolioNarratives)

성향 3종(방어/균형/공격) 각각에 대해 **구조화된 객체**로 작성.
줄글이 아니라 카테고리별 핵심 정보를 카드 형태로 정리.

⚠️ 비중은 시스템 기본값을 정확히 따라야 함 (변경 금지):
- 방어형: 주식 25% + 채권 55% + 대체 15% + 현금 5%
- 균형형: 주식 50% + 채권 25% + 대체 15% + 현금 10%
- 공격형: 주식 75% + 채권 0% + 대체 15% + 현금 10%

⚠️ 워딩 원칙 (절대 위반 금지):

[금지 표현]
- "균형 잡힌", "분산을 통해", "효과적으로", "안정적으로", "전반적으로"
- "위험자산 강세", "지정학 리스크" 같은 추상적 일반론
- 내가 만들어낸 표현, AI 분위기 나는 문장

[필수 표현]
- 실제 증권사 리서치 리포트에서 쓰는 워딩
  예: "캐리", "베이시스", "롱텀 듀레이션", "센티먼트", "롱숏", "엔케리 트레이드", 
       "스프레드 확대", "리스크 온/오프", "디스인플레이션", "드로우다운", 
       "테일 리스크", "헤징 코스트", "리얼 일드", "브레이크이븐 인플레이션"
- 뉴스 원문에 나온 정확한 인용 (인물명, 회사명, 발언 날짜 포함)
  예: "JP모건의 다이먼이 10/15 인터뷰에서 '채권 위기' 가능성을 거론"
  예: "버크셔 해서웨이가 3분기 13F에서 현금성 자산 3,970억 달러 사상최대"
  예: "한은 유상대 부총재의 11/2 발언 — 기준금리 인상 옵션도 열어둘 필요"
- 구체적 수치 + 변동률 모두 (예: "WTI $113.97, 6개월 +74.51%")

- 모든 수치는 소수점까지 정확히

# Part 6. 사용자 프로필별 추가 코멘트 (profileHints)

4가지 사용자 유형에 대해 각 150~200자, 자연스러운 줄글.
"~하라"가 아니라 "~하는 게 안전해 보입니다", "~해도 괜찮습니다" 같이 사람이 조언하는 톤.

1. "young_aggressive" — 20~30대, 공격적 성향, 장기 투자
2. "midage_balanced" — 30~50대, 자산 형성 중, 안정+성장
3. "retirement_defensive" — 은퇴 준비/은퇴자, 자산 보존 우선
4. "theme_focused" — 특정 테마(AI, 배당, 친환경 등) 집중 관심

각 유형별 작성 내용:
- 이번 달 시장에서 이 유형이 마주할 기회 1개 + 리스크 1개를 구체적으로
- 시장 환경 데이터·뉴스에서 직접 추론한 내용
- 일반론 금지. "~한 사람은 ~을 늘려도 됩니다" 같이 행동 조언으로 끝맺기
